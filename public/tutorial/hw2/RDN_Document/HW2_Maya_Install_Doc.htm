Maya 3.0 Tools – Installation for Homeworld 2

這份文件的目的是指導你設定萬艦齊發二RDN工具在Maya下工作。你必須先安裝好Maya 3.0或更新的版本。 注意：在這份文件釋出的時候，Maya的PLE版（個人學習版）並不能對應這些plug-in，和RDN工具並不相容。

（譯註：RDN官方工具內的plug-in據稱只能正常對應3.0版，能夠對應Maya5的Plug-in有國外玩家發佈在Relicnews討論版上。）

  1. 下載並安裝RDN工具包。
  2. 確認已經設定好環境參數：
    1. HW2_ROOT -> “C:\Program Files\Sierra\Homeworld2，或是你安裝萬艦齊發二的資料夾。
    2. MAYA_APP_DIR -> “%HW2_ROOT%\Maya”
    3. MAYA_INSTALL_DIR -> “C:\AW\Maya3.0”或是你安裝Maya的位置。
  3. 從[%HW2_ROOT%\Maya\3.0\plugins]把[IMFPSD.dll]複製到[%MAYA_INSTALL_DIR%\bin\plug-ins\image]，以讀取Photoshop的.PSD檔案。
  4. 從[%HW2_ROOT%\Maya\3.0\plugins]複製[msvci70.dll]到[%MAYA_INSTALL_DIR%\bin]，讓Maya能夠使用所釋出的plug-in。
  5. 如果你使用的Maya版本是4.0、4.5、或5.0，請執行下列的動作
    1. 將[%HW2_ROOT%\Maya\3.0]更名為[%HW2_ROOT%\Maya\#.#]，#.#是你的Maya的版本號碼(4.0、4.5、或5.0)。
  6. 如果你不確定環境變數是否都正常設定的話，當你第一次載入Maya的時候，開啟Relic Tool Shelf。
  7. 到Windows->Settings/Preferences->Plug-in Manager，並請確定下列的項目已經設為"auto load"（這些是萬艦二的plug-in，只要安裝設定正確就應該會出現）
    1. AnimExport.mll
    2. Annotation.mll
    3. FlightManeuversExport.mll
    4. FreezeShipScale.mll
    5. Homeworld2Export.mll
    6. HW2GetBB.mll
    7. LeverExport.mll
    8. Shizoom.mll
  8. 關閉並重新啟動Maya。
